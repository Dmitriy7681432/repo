Графическая сцена. Класс QGraphicsScene: сцена. Добавление компонентов на сцену

    На этом шаге мы укажем метод, используемые для выполнения этой операции.

    Помимо графических объектов, на сцену можно добавить компоненты, которые будут функционировать как обычно. Добавить компонент на сцену позволяет метод addWidget() класса QGraphicsScene. Формат метода: 

  addWidget(<QWidget>[, flags=0])

    В первом параметре указывается экземпляр класса, который наследует класс QWidget. Во втором параметре задается тип окна (см. 25 шаг). Метод возвращает ссылку на экземпляр класса QGraphicsProxyWidget, представляющий добавленный компонент.
