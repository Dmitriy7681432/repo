Обработка сигналов и событий. События окна (общие сведения)

    На этом шаге мы приведем общие сведения о событиях окна.

    Перехватывать все события следует только в самом крайнем случае. В обычных ситуациях нужно использовать методы, предназначенные для обработки определенного события, - например, чтобы обработать закрытие окна, достаточно переопределить метод closeEvent(). Методы, которые требуется переопределять для обработки событий окна, мы рассмотрим в следующих шагах.

    На следующем шаге мы рассмотрим изменение состояния окна.
